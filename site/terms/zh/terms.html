Npcink Cloud 服务条款

本服务条款适用于 Npcink Cloud 的访问和使用。Npcink Cloud 是面向 Npcink WordPress 插件的 hosted runtime 和服务端点。

生效日期：2026年6月1日
运营主体：麻城市牧泽世华工艺品
联系邮箱：npc@npc.ink
服务域名：https://cloud.npc.ink

服务

Npcink Cloud 提供 hosted runtime access、签名请求处理、health checks、entitlement summaries、媒体衍生处理、Site Knowledge 同步提示，以及可选的 metadata-only 插件监控能力。

只有当站点管理员配置 Cloud Base URL 和 Cloud API Key、验证连接，或本地 Npcink 组件显式调用 Cloud runtime client 时，才会使用 Npcink Cloud。

账户和 API Keys

访问 Npcink Cloud 可能需要账户以及由 Npcink Cloud 签发的 Cloud API Key。站点管理员负责妥善保管 API key，并在不再需要访问时移除访问权限。

Npcink Cloud Addon 插件会在 WordPress 服务器端保存 Cloud API Key secret，用于服务端请求签名。该 secret 不会打印到 wp-admin 页面。

本地 WordPress 控制权

Npcink Cloud 不拥有 WordPress 写入权限。最终 WordPress 写入、审批、proposal 记录、rollback、preflight 和本地治理仍由本地 WordPress 站点及其安装的 Npcink governance 组件负责。

Npcink Cloud 不作为 WordPress 控制面、WordPress 内的 billing truth、workflow engine、scheduler truth、prompt registry、router registry 或 approval system。

可接受使用

你不得使用 Npcink Cloud 传输违法内容、未经授权使用的凭据、恶意软件、缺乏合法基础或授权的敏感个人数据，或任何违反适用法律的内容。

可用性

Npcink Cloud 可能因维护、安全、滥用防范或运营原因变更、暂停或停止部分服务功能。

费用

部分 Npcink Cloud 功能可能需要付费计划或 entitlement。计划可用性、配额和限制可能因账户而异。

外部服务

Npcink Cloud 是由管理员配置的 WordPress 站点使用的外部服务。Npcink Cloud Addon 插件仅在管理员配置、验证、本地 Npcink 组件显式 runtime 调用，或管理员启用 monitoring 后联系配置的服务。
