Npcink Cloud 数据保留与删除

Npcink Cloud 仅出于运营、安全、调试、滥用防范、entitlement、会计和法律目的保留服务数据。

生效日期：2026年6月1日
运营主体：麻城市牧泽世华工艺品
联系邮箱：npc@npc.ink
服务域名：https://cloud.npc.ink

典型保留类别

  • 认证和请求签名元数据：用于安全和调试。
  • Runtime request 和 result metadata：用于服务运营和故障排查。
  • 可选 monitoring metadata：用于服务健康、可靠性、质量摘要和诊断。
  • Entitlement 和 usage summaries：用于账户状态、配额和服务访问。
  • Media derivative artifacts：根据其配置的过期时间或服务政策保留。
  • Site Knowledge change hints：仅在向 Cloud Site Knowledge 系统递送 public content-change hints 所需期间保留。

数据最小化

Npcink Cloud Addon monitoring 是 metadata-only，设计上不收集 prompts、article body content、generated content、raw payloads、provider credentials、Cloud API secrets、passwords、cookies、Authorization headers、database names、table names 或 filesystem paths。

删除

管理员可以通过 npc@npc.ink 请求删除与账户相关的 Npcink Cloud 服务数据。

部分记录可能因安全、滥用防范、会计、法律合规或备份完整性要求而继续保留。

联系

如对数据保留或删除有疑问，请联系：npc@npc.ink。
