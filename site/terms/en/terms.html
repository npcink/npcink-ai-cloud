Service

Npcink Cloud provides hosted runtime access, signed request handling, health checks, entitlement summaries, media derivative processing, Site Knowledge synchronization hints, and optional metadata-only plugin monitoring features.

Npcink Cloud is used only when a site administrator configures a Cloud Base URL and Cloud API Key, verifies the connection, or when a local Npcink component explicitly calls the Cloud runtime client.

Account and API Keys

Access to Npcink Cloud may require an account and a Cloud API Key issued by Npcink Cloud. Site administrators are responsible for keeping API keys confidential and for removing access when it is no longer needed.

The Npcink Cloud Addon plugin stores the Cloud API Key secret on the WordPress server for server-side request signing. The secret is not printed in wp-admin.

Local WordPress Control

Npcink Cloud does not own WordPress write authority. Final WordPress writes, approvals, proposal records, rollback, preflight, and local governance remain under the local WordPress site and its installed Npcink governance components.

Npcink Cloud does not act as a WordPress control plane, billing truth inside WordPress, workflow engine, scheduler truth, prompt registry, router registry, or approval system.

Acceptable Use

You may not use Npcink Cloud to transmit unlawful content, credentials that you are not authorized to use, malware, sensitive personal data unless you have a lawful basis and authorization, or content that violates applicable law.

Availability

Npcink Cloud may change, suspend, or discontinue service features for maintenance, security, abuse prevention, or operational reasons.

Fees

Some Npcink Cloud features may require a paid plan or entitlement. Plan availability, quotas, and limits may vary by account.

External Services

Npcink Cloud is an external service used by administrator-configured WordPress sites. The Npcink Cloud Addon plugin contacts the configured service only after administrator configuration, verification, explicit runtime use by a local Npcink component, or administrator-enabled monitoring.
